# PDF Layout Tool

このスキルは、PDF生成や印刷向けレイアウトを扱う機能で使う。

## 既定方針

- 要件とユーザー向け設定では mm を使う。
- PDFライブラリが必要とする境界でだけ point へ変換する。
- 仕様で別指定がない限り、A4 は 210 x 297 mm、A5 は 148 x 210 mm として扱う。
- 既定では画像のアスペクト比を保つ。crop/fill を提供する場合は挙動を明示する。
- ページ向き、余白、塗り足し、トンボ、並び順は仕様に明記する。

## 検証

- ページ数、ページサイズ、配置座標、画像順、奇数枚、回転をテストする。
- 可能なら固定サンプルを用意し、生成PDFを目視確認できるようにする。
- 印刷時に意図しない拡大縮小が起きないこと、余白とA判寸法が想定通りであることを確認する。
- 両面印刷や面付けでは、表裏順と見た目の配置を分けて記録する。
